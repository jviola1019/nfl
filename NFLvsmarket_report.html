How to read this report

Each row compares the blended model to the active moneyline market.

  • Blend Beat Market? shows whether the blend outperforms the market using the listed basis.
  • Basis tells you if the edge comes from final results, expected value, win probability, or a better price. Upcoming games rely on the pregame math.
  • Note summarizes the comparison so you can see how the edge was determined.
  • Δ columns flag any gap between the listed win percentage and the spread-implied win percentage.
  • Spread columns are presented from the home team's perspective; plus signs identify underdogs.
